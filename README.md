# zhou yu's blog

个人技术博客，基于 Jekyll + GitHub Pages 构建，在线地址：

[https://928979883.github.io/](https://928979883.github.io/)

## 本地开发

```bash
bundle install
bundle exec jekyll serve
```

访问 http://localhost:4000 即可预览。

## 技术栈

- Jekyll 静态站点生成器
- GitHub Pages 托管
- 自定义响应式主题，支持浅色 / 深色模式
- Simple Jekyll Search 站内搜索

## 目录说明

- `_posts/`：博客文章
- `_includes/`：页面片段
- `_layouts/`：页面模板
- `_sass/` / `style.scss`：样式文件
- `js/`：前端脚本
