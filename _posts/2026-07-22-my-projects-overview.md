这两年我参与了不少项目，技术栈从 Vue2 到 Vue3、React、Uniapp、微信小程序再到 Java 后端，覆盖能源、物流、政务、教育等多个行业。趁着整理个人仓库的机会，把做过的主要项目梳理一遍，也算一份个人作品集。

## 一、能源与工业类（主力方向）

我大部分精力投入在工业能源领域，做电能量采集、能耗监测、综合能源管理这类系统。

- **德扬能源官网 / 综合能源平台**：官网前台 Vue2 + Vue CLI，后台 Vue3 + Vite + Pinia + Element Plus + UnoCSS。后端是标准 Spring Boot 前后端分离架构。
- **南非 NECOS / 电能量采集 2025**：Vue2 老技术栈的采集系统，对接电表、负荷曲线。
- **焦煤供电 / 莆田八方码头 / 长水机场 / 湖南师大 / 海外 HES**：都是前后端分离的项目，做能耗监测、计量、告警、报表。
- **通用采集 - 综合能源**：可复用的采集与监控框架，适配多种协议和设备。
- **校园节水平台（威胜）**：节水平台，前端 Vue2。

## 二、大屏与可视化

- **AJ-Report 动态大屏（前后端）**：基于 AJ-Report 的数据可视化大屏，前后端都有涉及，适合做炫酷的指挥驾驶舱。
- **vue-mapvgl-dev**：基于 mapvgl 的地图可视化开发分支，做地理信息叠加。
- **郴州电量（建筑能耗，React）**：React 技术栈的建筑能耗展示。
- **拓扑图编辑器（个人）**：纯前端 Vue3 的项目，专门做网络/电力拓扑编辑，支持节点拖拽、连线、布局导出，是我个人打磨的一个小工具。

## 三、小程序与移动端

- **个人工具箱小程序（个人）**：微信小程序，记账、信息查询等日常工具，类目是工具 > 记账 / 信息查询。
- **郧西供水小程序**：供水相关的微信小程序。
- **一力物流园 App（Uniapp）**：用 Uniapp 做的物流园移动端。
- **旺远官网 Uniapp**：配套的移动端。

## 四、后端与中台

- **RuoYi-Vue + RuoYi（后端）**：基于 RuoYi 的权限中台，做快速业务搭建。
- **Feishu / 抖音工具（feishu-douyin-tool）**：飞书与抖音相关的自动化工具。
- **WeatherPush**：天气推送服务。
- **daily_reminder**：每日提醒小工具。

## 五、个人博客与写作

- **本博客**：Jekyll + GitHub Pages，记录学习与项目复盘，支持深色模式、阅读进度、目录导航。

## 一点感想

做过的项目虽多，但真正沉淀下来的个人作品不多。接下来打算把**拓扑图编辑器**和**个人工具箱小程序**这两个纯个人项目持续打磨，也把博客当成一个长期记录的地方。如果你也在做类似方向，欢迎在 GitHub 上交流。
