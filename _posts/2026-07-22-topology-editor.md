做电力和工业监控系统时，经常需要画网络拓扑、配电拓扑。市面上的工具要么太重、要么不开放导出。于是我动手做了一个轻量的**拓扑图编辑器**，纯前端、开箱即用。

## 技术选型

- **Vue 3 + Vite**：组合式 API，开发体验顺滑。
- **SVG / Canvas 渲染**：节点用 SVG，缩放平移性能好。
- **拖拽交互**：基于 Pointer 事件实现节点拖动、画布平移、滚轮缩放。
- **撤销重做**：维护操作栈，支持 Ctrl+Z / Ctrl+Y。

## 核心能力

1. **节点编辑**：支持自定义节点图标、名称、状态（在线/离线/告警）。
2. **连线**：点选端口即可连线，自动避让、贝塞尔曲线。
3. **布局**：提供网格对齐、自动分布。
4. **导出**：可导出 JSON（可再编辑）和 PNG 图片。
5. **示例数据**：内置一份电力监控示例，打开就能看到效果。

## 界面预览思路

编辑器左侧是节点面板，中间是画布，右侧是属性栏。整体走简洁深色工具栏风格，和工业监控大屏的调性一致。

```js
// 伪代码：添加一个节点
function addNode(x, y) {
  nodes.value.push({
    id: uid(),
    x, y,
    label: '设备',
    status: 'online'
  })
}
```
